# Vanilla Modal

Implementação de uma modal sem utilizar bibliotecas pré-prontas.

Funcionalidades

- [x] Responsiva 
- [x] Tema light / dark usando `prefers-color-scheme`
- [x] Com animação
- [ ] Tema light / dark com botão global
- [ ] Acessível
- [ ] Redução de movimentos com `prefers-reduced-motion`
